<p><ul><li>Random Access Memory</li><li>Read-only Memory</li></ul></p>
<p><p>Computer memory market consists of two main types - Random Access Memory (RAM) and Read-only Memory (ROM). RAM is volatile memory that temporarily stores data for faster access by the CPU. It is commonly used in computers for running applications and storing data during active use. On the other hand, ROM is non-volatile memory that stores essential system information and instructions that cannot be changed. Both types of memory play crucial roles in the overall performance and functionality of a computer system.</p></p>
